# HTML5 Aquarium Animation

A simple 2D aquarium animation built using HTML5 Canvas and JavaScript.

## Overview

This project was developed as part of a Computer Graphics course assignment. The animation simulates an underwater aquarium environment using basic geometric objects and animation techniques.

The scene contains several animated elements including fish, bubbles, seaweed, rocks, and underwater light effects to create a lively aquarium atmosphere.

## Features

* Multiple colorful fish swimming across the screen
* Rising bubble animation
* Animated seaweed movement
* Underwater lighting effect
* Responsive fullscreen canvas
* Smooth animation using `requestAnimationFrame()`

## Graphics Objects Used

| Object       | Shape           |
| ------------ | --------------- |
| Fish Body    | Ellipse         |
| Fish Tail    | Triangle        |
| Bubble       | Circle          |
| Rock         | Circle          |
| Seaweed      | Quadratic Curve |
| Light Effect | Line            |

## Animation Techniques

### Horizontal Translation

Fish move continuously from left to right across the screen.

### Vertical Translation

Bubbles move upward to simulate underwater air bubbles.

### Oscillation Motion

Seaweed moves periodically using a sine function to create a waving effect.

### Frame Animation

The animation is rendered continuously using JavaScript's `requestAnimationFrame()`.
